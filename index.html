Les selecteurs CSS

Selectors Appliqué à Exemple
Pseudo-classe :target e :target Tout élt HTML ciblé : a, h1, p,id, ... h1:target #texte1:target {color:red;}
de Type e Tout type d'élt HTML: a, p, h1, span, div, ... a {color:red;}
de Classe .c Tout élt ayant la class .rouge {color:red;}
e.c Tout élt ayant la class li.rouge {color:red;}
e.c.c Tout élt ayant à la fois les class li.extrait.rouge {color:red;}
ID # Tout élt ayant l'identifiant #demo {color:red;}
Universels * Tout élément HTML * [lang^=fr] {color:red;}
D'Attribut e[ attr. ] Cible tous les élts ayant l'attr. a[title] {color:red;}
e[ at="xxx " ] Elt avec un attr. qui correspond à xxx a[href="https://example.org"] {color:red;}
e[ at* ="xxx" ] Elt dont l'attr. contient une chaine de car. avec au moins 1 occurence de "xxx" a[href*="test"] {color:red;}
e[ at$ ="xxx" ] Elt dont l'attr. finit par "xxx" a[href$=".org"] {color:red;}
e[ at^ ="xxx" ] Elt dont l'attr. commence par le mot "xxx" exactement a[href^="test"] {color:red;}
e[ at~ ="xxx" ] Elt dont l'attr contient le mot "xxx" exactement suivi immédiatement d'un tiret (U+002D) a[href~="test"] {color:red;}
e[ at| ="x" ] Elt dont l'attr. contient le mot "test" exactement, séparé de blanc au sein d'une liste a[href|="test"] {color:red;}
e[ at="x" i ] Le i ou I permet de ne pas prendre en compte le casse. a[href="test" i] {color:red;}
Combinateurs e1 + e2 Uniquement aux élts (e2) qui « suivent » (e1) img + p {font-style: bold;}
e1 ~ e2 S'applique à tous les élts (e2) qui sont dans le premier élt (e1) img + p {font-style: bold;}
e1 > e2 Les élts (e2) qui sont enfants directs de (e1) ul.mon-truc>li {margin: 2em;}
